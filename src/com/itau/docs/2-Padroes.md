# Java Annotations

## Padrões (Built In)

A linguagem Java possui annotations padrões

* Estão dentro do pacote java.lang.package

* @Override
* @Deprecated
* @SafeVarArgs
* @SuppressWarnings
* @FunctionalInterface


## Meta Annotations

Annotation que são usadas dentro de outras annotations

* @Inherited
* @Documented
* @Target
* @Retention
* @Repeatale
